# MemoApp with react
- 왼편엔 sidebar 컴포넌트를 사용하여 메모 리스트가 출력되게, 오른편에는 react-quill을 사용하여 텍스트 작성이 가능함. 
- firebase 연동으로 페이지에 입력한 메모들은 cloud firebase에서 확인 가능.
